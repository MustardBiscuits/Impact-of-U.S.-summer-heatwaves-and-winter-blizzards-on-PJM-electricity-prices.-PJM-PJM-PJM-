Data folder guide
=================

This file explains how to obtain the data and where to place it inside the
data folder.

Folder structure
----------------

data/
  raw/
    PJM/
      pjm_dom_da_lmp_hrl_fixed.csv
      pjm_dom_load_hourly_2018_2025UTC.csv
    weather/
      weather_model_ready.csv
  processed/
    Generated model-ready files created by the Python pipeline.

How to obtain the data
----------------------

1. PJM day-ahead LMP data
   Obtain hourly PJM Dominion (DOM) day-ahead LMP data from PJM Data Miner 2 or
   another PJM data export. The required prepared file name is:

   data/raw/PJM/pjm_dom_da_lmp_hrl_fixed.csv

2. PJM load data
   Obtain hourly PJM Dominion load data from PJM Data Miner 2 or another PJM
   data export. The required prepared file name is:

   data/raw/PJM/pjm_dom_load_hourly_2018_2025UTC.csv

3. Weather data
   Obtain hourly weather observations for the PJM Dominion region and prepare a
   model-ready weather table with UTC timestamps and hourly weather features.
   The required prepared file name is:

   data/raw/weather/weather_model_ready.csv

Where to place the files
------------------------

Place the three raw input files exactly at the paths shown above. The pipeline
expects those names by default. If you use different names or locations, pass a
custom data root to:

python scripts/build_model_dataset.py --dataset-root /path/to/raw/data

Generated data
--------------

The data/processed folder is generated by the code. You do not need to download
or manually edit those files. Rebuild them with:

python main.py

or:

python scripts/build_model_dataset.py
